# Jack's Decorations
添加了几种创造模式对应方块的装饰版，具体如下：  
   * 装饰刷怪笼
   * 装饰屏障
   * 装饰基岩
   * 强化深板岩
   * 装饰命令方块

这些方块都需要使用下界合金镐才能有效挖掘。
