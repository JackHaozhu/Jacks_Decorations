# Jack's Decorations
添加了几种创造模式对应方块的装饰版，具体如下：  
   * 装饰刷怪笼
   * 装饰屏障
   * 装饰基岩
   * 强化深板岩
   * 装饰命令方块

注：本模组作用是添加外观与以上原版方块相同的方块，模组添加的方块不具备原版方块的功能。

这些方块都需要使用下界合金镐才能有效挖掘。  

计划：
   - 为装饰刷怪笼添加显示自选生物模型的功能
   - 为装饰屏障添加与原版屏障类似的手持现实功能
   - 修复装饰命令方块放置没有方向性的问题
